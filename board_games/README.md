# Board Games Collection

This folder contains various board-based games:

1. Chess
   - Classic chess game
   - AI opponent
   - Move validation

2. Checkers
   - Classic checkers game
   - Multiple players
   - King pieces

3. Battleship
   - Classic naval game
   - Multiple grid sizes
   - AI opponent

4. Monopoly (Simplified)
   - Property management
   - Multiple players
   - Chance cards

5. Scrabble
   - Word building game
   - Score calculation
   - Dictionary validation

Each game includes:
- Modern GUI interface
- Score tracking
- Multiple difficulty levels
- Sound effects
- Save/load functionality
- Leaderboards
