# Card Games Collection

This folder contains various card-based games:

1. Memory Card Game
   - Match pairs of cards
   - Multiple difficulty levels
   - Timer option

2. Blackjack
   - Classic casino game
   - Multiple players
   - Betting system

3. Solitaire
   - Classic card game
   - Multiple variants
   - Score tracking

4. Uno
   - Multiplayer card game
   - Special cards
   - AI opponents

5. Poker
   - Texas Hold'em
   - Multiple players
   - Betting system

Each game includes:
- Modern GUI interface
- Score tracking
- Multiple difficulty levels
- Sound effects
- Save/load functionality
- Leaderboards
