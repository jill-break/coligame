# Puzzle Games Collection

This folder contains various puzzle-based games:

1. Sudoku
   - Multiple difficulty levels
   - Hint system
   - Timer option

2. Tic Tac Toe
   - Classic game
   - AI opponent
   - Score tracking

3. Minesweeper
   - Classic puzzle game
   - Multiple grid sizes
   - First-click safety

4. 8-Puzzle
   - Sliding tile puzzle
   - Multiple sizes
   - Move counter

5. Connect Four
   - Classic game
   - AI opponent
   - Multiple players

Each game includes:
- Modern GUI interface
- Score tracking
- Multiple difficulty levels
- Sound effects
- Save/load functionality
- Leaderboards
